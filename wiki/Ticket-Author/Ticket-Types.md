# Ticket Types

- Epic
- Feature
- Spike
- Bug
- Chore
- Documentation
- ODD

Each type has a template under [`.claude/skills/ticket-author/assets/`](../../.claude/skills/ticket-author/assets/).

ODD tickets request resolution of an existing [Open Design Decision](../Wiki-Page-Author/Open-Design-Decision) on a wiki page. They exist so an ODD is assigned and visible to the discipline that owns the decision; they close when the wiki ODD is resolved per the [ODD resolution rules](../Wiki-Page-Author/Open-Design-Decision#resolving-an-open-design-decision). The ODD must already exist on a wiki page; the ticket does not create one.
